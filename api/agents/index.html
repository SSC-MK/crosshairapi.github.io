    AI Agents¶

    The Agents API provides autonomous AI agents that use the ReAct (Reasoning + Acting) pattern to complete multi-step tasks. Agents can plan, execute tools, observe results, and iterate until the task is complete.

    ReAct Pattern

    ReAct agents follow a Thought -> Action -> Observation loop:

    1. Thought -- The agent reasons about what to do next
    2. Action -- The agent selects and invokes a tool
    3. Observation -- The agent observes the result and decides the next step

    Endpoints¶

    List Agent Types¶

    GET /v1/agents

    List all available agent types and their capabilities.

    curl http://localhost:8001/v1/agents

    Run Agent¶

    POST /v1/agents/{agent_type}/run

    Launch an agent to perform a task asynchronously.

    Path Parameters:

    Name Type Description
    agent_type string Agent type (e.g., research, osint)

    Request Body:

    Field Type Default Description
    task string -- Natural language task description (required)
    max_steps integer 10 Maximum reasoning/action steps
    timeout integer 300 Task timeout in seconds 
    curl -X POST http://localhost:8001/v1/agents/research/run \
  -H "Content-Type: application/json" \
  -d '{
    "task": "Research the company Acme Corp and summarize their main products",
    "max_steps": 10,
    "timeout": 300
  }'

    curl -X POST http://localhost:8001/v1/agents/osint/run \
  -H "Content-Type: application/json" \
  -d '{
    "task": "Gather publicly available information about domain example.com",
    "max_steps": 15,
    "timeout": 600
  }'

    import requests
import time

# Start agent task
response = requests.post(
    'http://localhost:8001/v1/agents/research/run',
    json={
        'task': 'Research the company Acme Corp',
        'max_steps': 10,
        'timeout': 300
    }
)
task_id = response.json()['data']['task_id']

# Poll for results
while True:
    status = requests.get(
        f'http://localhost:8001/v1/agents/tasks/{task_id}'
    ).json()

    if status['data']['status'] in ('completed', 'failed'):
        break
    time.sleep(5)

print(status['data']['result'])

    List Agent Tasks¶

    GET /v1/agents/tasks

    List all agent tasks and their current status.

    curl http://localhost:8001/v1/agents/tasks

    Get Task Status¶

    GET /v1/agents/tasks/{task_id}

    Retrieve the status and results of a specific agent task.

    curl http://localhost:8001/v1/agents/tasks/task_abc123

    Cancel Task¶

    POST /v1/agents/tasks/{task_id}/cancel

    Cancel a running agent task.

    curl -X POST http://localhost:8001/v1/agents/tasks/task_abc123/cancel

    List Agent Tools¶

    GET /v1/agents/tools

    List all tools available to agents for use during task execution.

    curl http://localhost:8001/v1/agents/tools
